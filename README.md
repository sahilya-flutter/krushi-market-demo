#  Krushi Market - Agriculture Mobile Platform

Krushi Market is a real-world agriculture platform built to help farmers in Maharashtra with daily market prices, news, and useful updates.

---

##  Features

*  Daily Mandi Bhav (Market Prices)
*  Marathi Agriculture News
*  Firebase Push Notifications (FCM)
*  Weather Updates
*  Clean & Simple UI for Farmers
